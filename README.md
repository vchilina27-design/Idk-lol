# 3D Tank Battle (PC Version)

A modern 3D tank battle game implemented with Three.js. Fight alongside your teammates against enemy tanks in a tactical environment.

## Features

- **3D Graphics**: High-quality 3D models and lighting using WebGL.
- **Combat System**: Destructible enemies, armor management, and team-based warfare.
- **AI**: Intelligent teammates and enemies that track and engage targets.

## Controls

- **W / S**: Move Forward / Backward
- **A / D**: Rotate Tank
- **SPACE**: Fire Main Cannon

## How to Play on PC

1. Ensure you have **Python 3** installed.
2. Run the launch script:
   ```bash
   python3 launch_pc.py
   ```
3. The game will open in your default web browser at `http://localhost:8000`.

## Technical Details

The game is built using:
- **Three.js** (3D Engine)
- **HTML5 / JavaScript**
- **Tailwind-style CSS** for UI
